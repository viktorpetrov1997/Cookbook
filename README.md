# 📖 Cookbook
### 🧾 Project Description
Cookbook is a web application developed as part of the "JS Applications" course at SoftUni.
It allows users to create, browse, and search for recipes in a simple and interactive way.

The application supports both guest users and registered users. Guests can explore and search recipes, while registered users can fully interact with the platform by creating and managing their own recipes.
## ✨ Features
### 👀 Guest Users
* Browse all recipes
* Search for recipes
* View recipe details
* ❌ Cannot comment, create, edit, or delete recipes
### 👤 Registered Users
* 🔐 Register and log in
* 🍲 Create new recipes
* ✏️ Edit their own recipes
* 🗑️ Delete their own recipes
* 💬 Comment on any recipe
* 🔍 Browse and search all recipes
## 🛠️ Tech Stack
* HTML
* CSS
* JavaScript
* lit-html – for templating
* page.js – for client-side routing
## 🚀 Getting Started
#### 1. Clone the repository
```bash
git clone <repo-url>
cd cookbook
```
#### 2. Start the backend server
```bash
cd server
node server.js
```
#### 3. Start the frontend application
Open a new terminal window and run:
```bash
npm start
```
#### 4. Open the application
Once both servers are running, open your browser and visit:

👉 http://localhost:3000
